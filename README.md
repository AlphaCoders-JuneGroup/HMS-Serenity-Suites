# HMS - Serenity Suites

A comprehensive Hotel Management System developed for **Serenity Suites** to streamline hotel operations, improve guest experiences, and simplify administrative tasks through a centralized digital platform.

---

## Project Overview

HMS - Serenity Suites is designed to automate and manage the day-to-day operations of a hotel. The system provides efficient solutions for room reservations, guest management, billing, housekeeping, staff management, and reporting, enabling smooth hotel operations while enhancing customer satisfaction.

---

## Objectives

- Simplify hotel reservation management
- Improve guest service and experience
- Automate billing and payment processes
- Efficiently manage rooms and housekeeping
- Provide real-time operational insights through reports
- Reduce manual paperwork and administrative effort

## Installation

### Clone Repository

```bash
git clone https://github.com/AlphaCoders-JuneGroup/HMS-Serenity-Suites.git
```

### Navigate to Project

```bash
cd HMS-Serenity-Suites
```

### Install Dependencies

Backend

```bash
cd backend
npm install
```

Frontend

```bash
cd frontend
npm install
```

---

## Running the Project

Backend

```bash
npm start
```

Frontend

```bash
npm start
```
